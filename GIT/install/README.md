## GIT INSTALL , INITIALIZATION , USER DETAILS ON LINUX
-----------------------------------------------------------------


```sh
sudo su
yum install git -y 
git --version
mkdir mygit-linux
cd mygit-linux
git init
```


```sh
git config --global user.name "dev"
git config --global user.email "dev@gmail.com"
git config --list 
```
